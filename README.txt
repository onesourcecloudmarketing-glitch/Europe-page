OneSource Cloud - Private AI Infrastructure for Europe (v2)
==========================================================

Contents
- onesource-eu-private-ai-sitematch.html : the web page (now includes the
  OnePlus(TM) Platform section: Infrastructure Portal, PaaS Studio, Developer
  Hub, plus monitoring, usage analytics, cost reporting and audit).
- europe-map.svg : the Europe map shown in the hero (data centers: Amsterdam,
  Frankfurt, Marseille; Crimea shown as part of Ukraine).
- europe-map-preview.png : a preview image of that map.

IMPORTANT
Keep onesource-eu-private-ai-sitematch.html and europe-map.svg in the SAME
folder. The page loads the map from europe-map.svg via a relative path, so when
you publish, upload both files together.
